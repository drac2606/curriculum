Recomendación de uso de etiquetas semánticas

Es recomendable utilizar etiquetas semánticas como <header>, <nav>, <main>, <section>, <article>, <aside> y <footer> para estructurar el contenido de tu página web y mejorar la accesibilidad y el SEO. Más información en MDN Web Docs.

Lista ordenada de pasos para mejorar la semántica:

  1. Identificar las secciones principales del contenido.
  2. Utilizar etiquetas semánticas adecuadas para cada sección.
  3. Agregar descripciones y títulos claros.

Lista no ordenada de etiquetas semánticas útiles:

  • <header>
  • <nav>
  • <main>
  • <section>
  • <footer>
